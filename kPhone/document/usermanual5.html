The Active-Call-Field

The active call fields serves two purposes:

  • to display the calls state

  • to control the call

The functions:

1. Double Click: The CallWidget pops up or disappears if it was visible

2. Right mouse button click: a popup with five entries appears:
  • toggle hold, see chapter Call Hold
  • refer see chapter Call Transfer and Refer
  • transfer, see chapter Call Transfer and Refer
  • hang up, release the call
  • toggle CallWidget ,the CallWidget pops up or disappears if it was visible
