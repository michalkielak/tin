KPhoneIS User Manual: Messaging

This feature uses the SIP-signaling resources. This may stress the proxy and thus the proxy may discard or shorten messages! A more reliable messaging tool is described later.


The user may start messaging by pressing the right mouse button in the contact field and then selecting the Send Message field. If the mouse is clicked over an existing contact, it is selected, otherwise the user must enter an URI himself.

The Chat Window:



  • Remote

    Enter or change the destination (Partner-URI)

  • Status holds the message flow

  • Message

    type and send messages.
