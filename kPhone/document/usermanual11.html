The Dialer is a little remote dial command line application, start it with
   kpsidial URI
or
   kpsidial type URI
  
URI is your sip phonenumber with or without domain part
type is a one char string:

type
 meaning
a audio
v audio+video
o video only
1 first external session type
2 second external session type
3 third external session type
4 fourth external session type


For URI and the sessiontypes and how to handle them, see "KPhoneIS User Manual:  How to Call"

Activation

The dialer is initially inactive. You activate it from

 Preferences->SipPreferences->Settings->Remote Dialer.

It is necessary to restart kphone after the change.
 

If the dialer fails

clear the dialer's socket  by hand:

    rm /tmp/kpsidg

Dialer and Mozdev

Seamonkey


mozdev is an extension of Firefox and Seamonkey. Among other capabilities, it integrades external tools
into the browser allowing them to handle links. Although "sip" is no recognized link type yet, mozdev may be
used utilizing the "universal handler" for it.

In your Seamonkey go to Preferences->modzdev and
set below "Universal Handler" the "Scheme" to sip and  "Command" to /usr/local/bin/kpsidial. The path to kpsidial
may be different on your box.

Firefox

In your Firefox go to Extras->Add-ons. In the line Mozexx press Preferences . A new window will pop up,
where you should fill sip as protocol and select kpsidial as Command.
