Presence Modes


KPhone SI allows two presence modes:
  • End-to-end mode , in this case KPhoneSI  acts Edge Presence Server (sends  and receives SUBSCRIBE  and NOTIFY)
  • Presence server mode, KPhoneSI sends PUBLISH  and SUBSCRIBE messages to the presence  server and receives NOTIFY from  that.
To  set KPhone into the end-to-end mode, in Preferences->Settings set "Expiry Time of Publish" to zero.
