      This extension provides voice encryption based on the SRTP protocol.
     
    

Configuration:

     The configuration is done via the GUI:
  •      Select Preferences->SIP Preferences->SRTP)
  •       - select the "SRTP Encryption" toggle button
  •       - enter your SRTP Master key
  •       - Press OK

Use

You and your callee have to share the same encryption key, if you call a partner with different key,
one has to change the key. Calls to unencrypted partners are released immediately.
